Дата выполнения: 2025 год

1. Введение

Целью данной лабораторной работы являлась разработка интерактивного веб-приложения для трёхмерного моделирования и конструирования ламп. Приложение позволяет пользователям создавать уникальные дизайны ламп, настраивая параметры абажура, подставки и соединительных элементов в реальном времени.

Проект демонстрирует возможности современных веб-технологий для работы с 3D-графикой, включая библиотеку Three.js, WebGL для аппаратного ускорения рендеринга, а также интерактивный пользовательский интерфейс.

2. Цель и задачи работы

Цель:

Разработка полнофункционального веб-приложения для генерации и визуализации 3D-моделей ламп с возможностью экспорта в формат STL.

Задачи:

  • Реализация 3D-редактора с использованием Three.js
  • Создание параметрической системы настройки компонентов лампы
  • Разработка интуитивно понятного пользовательского интерфейса
  • Реализация функции экспорта моделей в формат STL для 3D-печати
  • Добавление режима тестирования освещения в различных средах
  • Обеспечение поддержки undo/redo для истории действий

5. Режимы работы

5.1. Режим редактирования

Основной режим работы, в котором пользователь может настраивать все параметры лампы. Интерфейс разделён на три вкладки: «Абажур», «Подставка», «Соединение». Каждая настройка мгновенно отображается в 3D-просмотре.

5.2. Режим тестирования

Специальный режим для визуализации лампы в различных условиях освещения:

  • Ночь — тёмная комната с активным источником света
  • Вечер — приглушённое окружающее освещение
  • День — яркое естественное освещение
  • Студия — профессиональное студийное освещение

В режиме тестирования доступна настройка цвета лампы, яркости, размера источника света и его положения относительно абажура.

5.3. История действий

Реализована система undo/redo с хранением до 50 последних состояний. Пользователь может отменять и повторять свои действия с помощью кнопок «←» и «→».

6. Экспорт моделей

Приложение поддерживает экспорт созданной модели абажура в формате STL (Stereolithography) — стандартном формате для 3D-печати. Экспортируется только геометрия абажура, так как именно эта часть наиболее интересна для изготовления на 3D-принтере.

function downloadSTL() { const shadeMesh = allMeshes.find(m => m.geometry); if (!shadeMesh) { alert('Сначала создайте абажур!'); return; } const exp = new STLExporter(); const res = exp.parse(shadeMesh, { binary: true }); const blob = new Blob([res], { type: 'application/octet-stream' }); const a = document.createElement('a'); a.href = URL.createObjectURL(blob); a.download = 'lampshade.stl'; a.click(); }

7. Архитектура приложения

7.1. Структура кода

  • init() — инициализация сцены, камеры, рендерера и контролов
  • updateAll() — обновление всех компонентов лампы
  • buildLamp() — построение 3D-модели лампы
  • getShapeProfile() — генерация профиля формы абажура
  • createHexGeo() — создание шестиугольной геометрии
  • addPleats() — добавление складок к абажуру
  • getShadeMat() — создание материала абажура
  • applyPattern() — применение узоров к текстуре
  • pushHistory()/undo()/redo() — управление историей

7.2. Компоненты 3D-сцены

  • Scene — основная сцена Three.js
  • Camera — перспективная камера с углом обзора 45°
  • Renderer — WebGL-рендерер с поддержкой теней
  • OrbitControls — управление вращением и зумом камеры
  • Lighting — комбинация AmbientLight и DirectionalLight
